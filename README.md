🚀 Built **CodeLab – A LeetCode-inspired Coding Platform using ReactJS**

CodeLab is a **lightweight competitive programming platform** designed to simulate the core experience of coding practice websites like LeetCode, where users can solve algorithmic problems, write code in multiple programming languages, and participate in basic contest-style challenges. The main focus of this project was to build a **clean, responsive, and structured frontend experience** that mirrors real-world coding platforms.

🔹 **Key Features:**

* Secure **User Authentication system (Login/Register)**
* Organized **problem listing with structured difficulty levels (Easy/Medium)**
* Interactive **multi-language code editor** for writing and submitting solutions
* Submission workflow with result feedback (Accepted / Wrong Answer / Runtime simulation style handling)
* Basic **contest simulation with leaderboard UI**
* User dashboard to track submissions and activity history
* Clean navigation with protected routes for authenticated users

🔹 **Tech Stack:**
ReactJS, React Router, Context API (or Redux if applicable), JavaScript, HTML5, CSS3

🔹 **Key Learnings:**

* Designing and structuring scalable React component architecture
* Managing complex UI state for a coding platform-like workflow
* Implementing routing and protected routes in React applications
* Building a real-world inspired UI/UX for developer-focused platforms
* Understanding how coding platforms structure problem-solving pipelines

This project helped me gain practical exposure to how competitive programming platforms are structured on the frontend side and improved my ability to design **user-focused, functional web applications**.
